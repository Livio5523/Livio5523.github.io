Archipel

This project was carried out during the preparatory year at ECAL as part of the colour course. We had to use a certain paper but we were free for the rest.


So I decided to create an augmented reality augmented reality application using the UNITY game engine. I also used Blender for the 3D models.


I created a system of interactive maps so that players could create their own archipelago.
